== Topics

This module is designed to give students a high level overview of computer
architecture and operating systems. We will be using the computer's file system
heavily over the semester and it is important to understand what it is and what
it is for.

== Learning Outcomes

After completing this module, students should:

- Describe the purpose of a computer operating system.
- Locate and use their computer's file system.
